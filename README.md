# ToDo List API - .NET Minimal API

This is the backend server for a full-stack ToDo List application. The project focuses on building a lightweight, high-performance API that bridges a React frontend with a cloud-hosted MySQL database.

## üöÄ Key Features
* **Minimal API Architecture:** Developed using .NET 8 for a clean and efficient codebase.
* **Database Integration:** Connected to a remote **MySQL** database (hosted on CleverCloud) using **Entity Framework Core** (DB First approach).
* **Full CRUD Support:** Endpoints for fetching, adding, updating, and deleting tasks.
* **API Documentation:** Fully documented with **Swagger UI** for easy testing and integration.
* **Deployment:** Hosted on **Render** using a **Docker** container.
